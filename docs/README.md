### <font color="orange">Круг</font>

[Ссылка на вики](https://ru.wikipedia.org/wiki/Окружность)
- Площадь: $S = \pi r^2$
- Периметр: $P = 2\pi r = \pi d$

Example: `area(5) = 78,5`

### <font color="green">Квадрат</font>

[Ссылка на вики](https://ru.wikipedia.org/wiki/Квадрат)
- Площадь: $S = a * a$
- Периметр: $P = 4 * a$

Example: `area(5) = 25`
